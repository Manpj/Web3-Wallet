# bip39（比特币的BIP-39标准）介绍
## 生成助记词
1、创建一个128比特或256比特的密码学强度的随机数S.
2、取S的哈希值的前（S的长度/32）比特，作为S的校验值.
3、S+校验值（P）.
4、以11比特为单位，将P分成多个组.
5、映射为单词.
6、保持顺序，即生成助记词.

## 助记词到种子密钥
1、助记词+盐（可选密码）
2、使用2048次哈希运算
3、生成512比特种子
